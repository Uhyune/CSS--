CSS display: block, inline, inline-block

div 태그는 block 속성

block 속성은 브라우저 좌측부터 우측 끝까지

block 속성은 크기를 가짐 - width height 지정

div는 block 속성
===일반 텍스트 =====

span 태그는 in line 속성

inline 속성은 한 라인안에서 영역을 차지

inline 속성은 내용의 크기로 정해짐

width height 지정해도 크기 변하지않음

span 태그는 in line 속성이다 그래서 한줄에 표시 크기를 지정해도 크기가 달라지지않음

inline-block 속성

inline-block 속성은 한 라인내에서 영역을 차지 (inline)

inline-block 속성은 크기를 지정하면 적용됌(block)

inline-block은 전체영역을 차지하지 않고 한 줄에 표시되지만,
가로, 세로 크기를 가짐
